# RunBird — Project Brief

## Магазин
Интернет-магазин кастомных handmade аксессуаров для **Chevrolet Corvette**.
Все продукты производятся вручную под заказ (made-to-order).
**Рынок:** США. **Платформа:** CS-Cart.

---

## Бренд

**Название:** RunBird / TheRunBird
**Сайт:** https://therunbird.com/
**Слоган:** FEEL THE DRIVE
**Логотип:** `Лого дискриптор_Б.svg` (inline SVG, цвет `#f1ebdb`)

### Логотип
- Кастомный bold wordmark "RunBird" с декоративным swoosh-элементом (силуэт птицы/скорость)
- Цвет: кремовый `#f1ebdb` — идеален на тёмном фоне
- Слоган "FEEL THE DRIVE" встроен в SVG (нижняя строка)
- Использовать как inline `<svg>` для масштабирования и смены цвета через CSS

## Дизайн-система (тёмная тема)

```
Фон:        #080808
Card:        #111111
Elevated:    #161616
Accent (A):  #C8A84C  (gold)
Accent (B):  #CC0000  (red)
Text:        #F0F0F0 / #808080 / #444444
Border:      rgba(255,255,255,0.07)
```

**Шрифты:** Barlow Condensed (заголовки) + Inter (текст)
**Текстура:** Carbon fiber — repeating-linear-gradient 45°/-45°
