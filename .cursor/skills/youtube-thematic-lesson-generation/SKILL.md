# 🎬 YouTube Thematic Lesson Generation Skill

## Overview

Этот скил позволяет быстро генерировать профессиональные постеры для тематических уроков на YouTube. Использует **ТОЛЬКО оригинальные активы** (фото Дмитрия Калашника и логотип БНЕЙ НОАХ).

**CRITICAL RULE**: Всегда ссылайтесь на `youtube-thematic-lesson-originals-rule.mdc` перед генерацией!

---

## 📐 Типы постеров

### 1️⃣ ТЕМАТИЧЕСКИЙ УРОК (ОСНОВНОЙ ТИП)

**Описание**: Темно-синий фон, золотые линии, фото слева в геометрической рамке

**Элементы**:
- Логотип БНЕЙ НОАХ (вверху слева)
- "◆ ТЕМАТИЧЕСКИЙ УРОК ◆" (золотой текст, вверху справа)
- Основной вопрос/тема (большой белый текст, 48-56px)
- Подзаголовок (белый текст, 32-40px)
- Источник (золотой текст, 24-28px)
- Имя лектора "Дмитрий Калашник" (внизу белый)
- Фото Дмитрия (слева, в золотой геометрической рамке)

**Фон**: Темно-синий (#1a3a7a)
**Размер**: 1920x1080px

**Примеры** (видны в примерах):
- "Почему об иудаизме нельзя судить по евреям?"
- "Почему разведчики (мерагльим) перечислены без особого порядка?"
- "Как посмел Егошуа затыкать рот пророкам Эльдаду и Мейдаду?"

---

## 🎨 ПАРАМЕТРЫ ГЕНЕРАЦИИ

### Обязательные параметры:

```
main_question: str          # Основной вопрос (48-56px, белый, максимум 2-4 строки)
subtitle: str               # Подзаголовок/дополнение (32-40px, белый)
source: str                 # Источник/глава/книга (24-28px, золотой)
lesson_type: str            # тип урока (например: "thematic", "chapter", "special")
photo_code: str             # Код фото (MK1_0393-2, MK1_0393, MK1_0395 и т.д.)
```

### Опциональные параметры:

```
subtitle_color: str         # Цвет подзаголовка (по умолчанию: белый)
use_watermark: bool         # Использовать еврейский текст водяным знаком (по умолчанию: True)
frame_style: str            # Стиль рамки вокруг фото (trapezoid, rectangle - по умолчанию: trapezoid)
```

## 💡 СОВЕТЫ И ТРЮКИ

### Совет 1: Выбор фото
- MK1_0393-2: Лучший для основных уроков (фон библиотеки ярче)
- MK1_0393: Альтернатива MK1_0393-2
- MK1_0395: Более строгий вид
- MK1_0392: Мягче, более дружелюбно
- MK1_0389: Немного другой угол

### Совет 2: Длина вопроса
- Оптимально: 2-3 строки
- Максимум: 4 строки
- Минимум: 1 строка (но выглядит неполно)

### Совет 3: Подзаголовок
- Может быть подвопросом ("Почему?", "Как?")
- Может быть дополнением ("Тайны...", "Секреты...")
- Может быть указанием автора или источника

### Совет 4: Источник
- Обычно: "Глава [название]" или "Книга [название]"
- Может быть: "Талмуд, Трактат [название]"
- Может быть: "Мидраш [название]"
