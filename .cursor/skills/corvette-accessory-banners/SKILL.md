# Corvette Accessory Banner Generator

## Overview

This skill helps create professional, consistent product category banners for Chevrolet Corvette C6 accessories. All banners use the same template styling with a unified Corvette interior background and consistent footer design for category names.

## Banner Template Specifications

**Dimensions**: 800x430 pixels (or proportionally larger, e.g., 1600x860, 2400x1290)

**Design Elements**:
- Corvette C6 luxury interior background (dark, professional automotive setting)
- Premium accessory component centered in upper portion
- Unified dark gray/charcoal footer bar at bottom (75px height)
- Professional dramatic lighting with smooth gradients
- Realistic high-quality automotive product photography style
- No text or logos on the main image area

**Layout Structure**:
- **Upper section** (355px for 800x430): Accessory showcase with Corvette interior background
- **Lower section** (75px): Solid dark gray/charcoal bar for category name text overlay
- Seamless transition between product area and text footer

## Creating a Banner

### Essential Information to Gather

Before generating a banner, collect:

1. **Accessory type** - What Corvette C6 component (handbrake, shift knob, etc.)
2. **Material/variant** - Premium material, color, or specific version
3. **Style preference** - Realistic product photo, angle, emphasis details
4. **Category name** - Text that will go in the footer (for reference only)

### Generation Process

When creating a banner:

1. **Identify the accessory** and its key selling points (premium materials, craftsmanship, details)
2. **Use the GenerateImage tool** with these parameters:
   - Description: Include accessory type, material details, premium aspects, and background reference (Corvette C6 interior)
   - Specify the 800x430px dimensions or larger proportional size
   - Specify the 75px footer bar specification
   - Filename: Use descriptive naming (e.g., `corvette_accessory_[component_name]_banner.png`)

3. **Verify the footer** is consistent with other banners before finalizing

### Footer Design Consistency

**Always maintain**:
- Dark gray/charcoal color (#3a3a3a to #4a4a4a range)
- 75px fixed height in all banners
- Positioned at the very bottom
- Smooth, professional appearance
- Ready for white or light text overlay

### Background Consistency

**Corvette Interior Background**:
- Dark luxury automotive setting
- Corvette C6 interior elements visible (steering wheel, dashboard, carbon fiber, chrome details)
- Professional dramatic lighting
- Dark gradient or neutral tone
- Emphasizes premium quality of accessories
