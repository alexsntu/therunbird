- **Размер**: 1080x1080px (квадратные карточки)

## ТЕХНИЧЕСКИЕ ДЕТАЛИ

### Размер фото для reference
- Использовать фото с интернета: один ручник с разными материалами для reference

### Процесс генерации
1. Найти/создать reference фото с разными вариантами материала одного продукта
2. Использовать это фото через `reference_image_paths` в GenerateImage
3. Генерировать каждую вариацию с уточнением материалов в description

### Палитра цветов
- **Карбон**: Черный глянец с серебристыми бликами
- **Кожа**: Коричневая, черная, премиум
- **Алькантара**: Темно-серая, черная, матовая текстура велюра
- **Перфорация**: Видимая, мелкая, регулярная
- **Красный карбон**: Ярко-красный с углеродным переплетением

## ИНСТРУКЦИИ ДЛЯ СЛЕДУЮЩИХ СЕССИЙ

1. Начать с категории 2 (Standard Gear Shift Knob)
2. Найти или создать reference фото стандартной ручки КПП с показанием разных материалов
3. Использовать то же оформление: логотип RUNBIRD, английские названия, 1080x1080px, темный фон
4. Постоянно проверять, что все 7 вариаций одной категории имеют ОДИНАКОВУЮ ФОРМУ, только разные материалы
5. Сохранить reference фото в папке /assets для использования в следующих категориях

## ВАЖНЫЕ ПРИМЕЧАНИЯ

- ❌ НЕ делать салонные вариации (in-car shots) - только премиум студийные
- ❌ НЕ менять форму продукта между вариациями - только материалы
- ✅ ВСЕ тексты на английском языке
- ✅ ОДИН логотип RUNBIRD на всех карточках
